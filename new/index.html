Senior Software Developer

I've been developing services and applications for years, mostly for SAP. After working on the CloudFoundry offering that enabled teams to developing cloud-native applications, I dedicated my work onto deploying fully-managed Application Lifecycle Management on top of Kubernetes, I know spend my time as an application developer to build SAP's Data Warehouse Cloud.

Data Warehouse Cloud

SAP Data Warehouse Cloud is a software-as-a-service Business Warehouse offering. It is an integrated, fully managed service, and persona-driven Data Warehouse as a Service solution, suitable for SAP and non-SAP customers offering reduced deployment complexity, flexible pricing with integration to SAP Intelligent Enterprise Suite solutions, SAP Analytics Cloud, SAP Cloud Platform services, Partner solutions and open-source technologies.

Gardener

Gardener is an open source project driven by SAP to create, update and monitor kubernetes clusters. Gardener can operate hundrets of clusters with operation, management and monitoring capabilities. I've initiated a project, now called Potter, automating an fully managing installed software and services on top of those clusters. With that, stakeholders can't only get fully managed clusters, by the software on top of it are install- and opererateable in an "off-the-shelf" fashion.

Peripli

Peripli, or the Service Manager enables service consumption across different platforms. Platforms need to implement the Open Service Broker Api and can then consume a service running in a remote platform. In a team I was developing this service brokner, enabling cross consumption between the different platform stacks, Neo, CloudFoundry and Kuberentes, at SAP.

ReviewNinja

"ReviewNinja is a light-weight code-review tool which fits neatly into the GitHub pull request workflow. It presents all the project information needed to commit a pull request and merge code confidently." [ReviewNinja official]. The tool was developed before GitHub Pull requests existed and provided line-by-line codereviews on top of the platform.

CatchyTune

CatchyTune is an online music player where artists can share their music and their fans listen to it. The project was created as part of a Software Engineering course at Duale Hochschule Karlsruhe. As it provided streaming of music and upload functionality, I am not hosting it anymore, for legal reasons. Also this was my first and only RoR project - the speed of initial development and prototyping was a blast, and is certainly the reason for the project being well-graded. Thanks David!

XP days 2017

I went to the XP days 2017 in Stuttgart. The content reached from practical teamwork by building a event hall with LEGO to practice agile methodologies over to theoretical lectures on how to push change as a developer in large organizations.

Object Calisthenics Validator

Jeff Bay's "Object Calisthencs" are a short coding challenge described in The ThoughtWorks Anthology. I have created an Eclipse plugin to validate if code created during the training, satisfies the rules. The tool adds hints and warnings to the respective code line. As part of a student research paper I have furthermore analyzed how such a validation for each rule could be implemented.

Personal

I am an avid diver in my freetime. Re-learned proper diving only after diving with Global Underwater Explorers, the diving association that formed out of the activities Woodville Karst Plain Project.
