ReviewNinja

2014

JavaScript, AngularJS, NodeJS, Socket.io, MongoDB

ReviewNinja is a light-weight code-review tool which fits neatly into the GitHub pull request workflow. It presents all the project information needed to commit a pull request and merge code confidently.

It is currently developed by the internal tools team of the SAP Labs in Palo Alto and the largest open source project driven by SAP. Our source code is on github.com/review.ninja/review.ninja. Feel free to contribute!

CatchyTune

2013

Ruby, Ruby on Rails

CatchyTune is a online music streaming solution, developed as part of an University Project. The music platform enables creative, independant artists to upload their music for free and give an opportunity to fans to listen to it.

This way music fans from around the world can easily play the music they love from the internet. CatchyTune offers a search functionality and a playlist. The project was developed within 2 months including customer surveys, customer feedback, UI/UX-Design, Project Management and implementation.
