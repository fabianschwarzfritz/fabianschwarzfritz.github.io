I've been developing services and applications for SAP for years. After working on the CloudFoundry offering that enabled teams to developing cloud-native applications, I am now dedicating my work onto deploying kubernetes and integrating it into SAP's ecosytem.

Data Warehouse Cloud

SAP Data Warehouse Cloud is a software-as-a-service Business Warehouse offering. It is an integrated, fully managed service, and persona-driven Data Warehouse as a Service solution, suitable for SAP and non-SAP customers offering reduced deployment complexity, flexible pricing with integration to SAP Intelligent Enterprise Suite solutions, SAP Analytics Cloud, SAP Cloud Platform services, Partner solutions and open-source technologies.

Gardener

Gardener is an open source project driven by SAP to create, update and monitor kubernetes clusters. Gardener can operate hundrets of clusters with operation, management and monitoring capabilities. I am in the team that integrates these kubernetes clusters into the SAP ecosystem and enables to use services on top, which are useful in the SAP context.

Peripli

Peripli, or the Service Manager enables service consumption across different platforms. Platforms need to implement the Open Service Broker Api and can then consume a service running in a remote platform. From a developer's perspective the service feels like it was integrated directly into the local platform.

ReviewNinja

"ReviewNinja is a light-weight code-review tool which fits neatly into the GitHub pull request workflow. It presents all the project information needed to commit a pull request and merge code confidently." [ReviewNinja official].

CatchyTune

CatchyTune is an online music player where artists can share their music and their fans listen to it. The project was created as part of a Software Engineering course at Duale Hochschule Karlsruhe.

XP days 2017

I went to the XP days 2017 in Stuttgart. The content reached from practical teamwork by building a event hall with LEGO to practice agile methodologies over to theoretical lectures on how to push change as a developer in large organizations.

Object Calisthenics Validator

Jeff Bay's "Object Calisthencs" are a short coding challenge described in The ThoughtWorks Anthology. I have created an Eclipse plugin to validate if code created during the training, satisfies the rules. The tool adds hints and warnings to the respective code line. As part of a student research paper I have furthermore analyzed how such a validation for each rule could be implemented.
