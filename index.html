Object Calisthenics Validator

Object Calisthenics was introduced Jeff Bay in The Thoughtworks Anthology published by The Pragmatic Programmers. The calisthenics consist of nine rules that are applied in a short coding challenge. The training focuses on Object Oriented principals. During the training the developer has to verify for himself if the his code adheres to the rules. As part of a student research paper, I analysed the rules and implemented a tool to validate the rules. The eclipse plugin adds hints and warnings to each line where the rules are broken.
