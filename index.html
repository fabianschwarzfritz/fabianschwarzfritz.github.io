Projects at Work

Data Warehouse Cloud

SAP Data Warehouse Cloud is a software-as-a-service Business Warehouse offering. I technically lead a team and we brought concept to production of the data warehouse task management system in 6 months. The system is used by 200+ customers for production data warehousing and manages 86.000 data long-running data replication tasks per day. Over the three years the software has grown and now supports features like cloud-native resiliency and task chains. I helped bringing technical concepts and enginneering approaches from the cloud-native to SAP's business warehouse world.

Gardener

Gardener is an open source project driven by SAP to create, update and monitor kubernetes clusters. Gardener can operate hundrets of clusters with operation, management and monitoring capabilities. I've initiated a project, now called Potter, automating an fully managing installed software and services on top of those clusters. Stakeholders can define their software bill-of-materials and the hub deploys and reconciles packages on the cluster. Besides technical CRDs, a user friendly UI helps exploring available packages and monitoring what's running on the cluster.

Peripli

Peripli, or the Service Manager enables service consumption across different platforms. Platforms need to implement the Open Service Broker Api and can then consume a service running in a remote platform. In a team I was developing this service brokner, enabling cross consumption between the different platform stacks, Neo, CloudFoundry and Kuberentes.

Object Calisthenics Validator

Jeff Bay's "Object Calisthencs" are a short coding challenge described in The ThoughtWorks Anthology. I created an Eclipse plugin to validate if code created during the training, satisfies the rules. The tool adds hints and warnings to the respective code line. As part of a student research paper I have furthermore analyzed how such a validation for each rule could be implemented.
