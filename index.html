Peripli

Peripli, or the Service Manager enables service consumption across different platforms. Platforms need to implement theOpen Service Broker Apiand can then consume a service running in a remote platform. From a developer's perspective the service feels like it was integrated directly into the local platform.

Object Calisthenics Validator

Jeff Bay 's "Object Calisthencs" are a short coding challenge described inThe ThoughtWorks Anthology. I have created an Eclipse plugin to validate if code created during the training, satisfies the rules. The tool adds hints and warnings to the respective code line. As part of a student research paper I have furthermore analyzed how such a validation for each rule could be implemented.
